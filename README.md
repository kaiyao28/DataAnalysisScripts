# DataAnalysisScripts

A personal reference library of reusable data analysis scripts.

## Python (`python/`)

| File | Description |
|------|-------------|
| `01_cleaning.py` | DataFrame creation, missing values, duplicates, dtypes, outliers, string cleaning |
| `02_exploration.py` | EDA — describe, value counts, distributions, correlations, quick plots |
| `03_wrangling.py` | Filter, sort, groupby, merge/join, pivot/melt, datetime, new columns |
| `04_numpy.py` | Array creation, indexing, math, statistics, linear algebra |
